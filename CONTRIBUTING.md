# 贡献指南 | Contributing Guide

感谢您对本项目的兴趣！

## 与上游项目的关系

本项目是 [TetherIA/aero-hand-open](https://github.com/TetherIA/aero-hand-open) 的衍生项目，专注于 WebSocket 通信扩展。

## 贡献类型

### 🐛 Bug 修复
- 请提交 Issue 描述问题
- 附上复现步骤和环境信息

### ✨ 新功能
- 建议先提交 Issue 讨论
- 功能需与 WebSocket 通信相关

### 📖 文档改进
- 欢迎提交文档修正或补充

## 贡献流程

1. **Fork** 本仓库
2. 创建特性分支：`git checkout -b feature/your-feature-name`
3. 提交更改：`git commit -m 'Add some feature'`
4. 推送分支：`git push origin feature/your-feature-name`
5. 创建 Pull Request

## 代码规范

- Python：遵循 PEP 8
- C：遵循 K&R 风格
- 提交信息：使用清晰的英文描述

## 许可证

通过贡献本项目，您同意您的代码采用 Apache-2.0 许可证。
